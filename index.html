Docker Compose 配置转换工具

|

将 Docker Run 命令或 Docker Compose 配置转换为标准 YAML 文件

输入 Docker Run 命令 / Docker Compose 配置

支持 Docker Run 命令（含 docker pull）、多行命令（带 \ 或漏写 \）、或完整的 Docker Compose YAML 配置
· 留空则不做任何替换
· $(pwd) 等 shell 变量会被自动去掉
· {container_name} 会自动替换为实际容器名
· 系统级挂载如 /var/run/docker.sock 等不会被替换
· 默认值从 index.html 同目录下的 volumes.js 读取，修改后刷新生效
仅在命令中未显式指定对应参数时生效
